## 10.4 关键招聘标准

### 产品经理
- 3年以上B2B SaaS产品经验
- 至少1个行业（餐饮/美业/教育/零售）深度理解
- 数据驱动决策习惯
- 具备技术背景优先

### 全栈工程师
- 3年以上Node.js/Python经验
- 小程序开发经验（微信/小红书至少1个）
- 熟悉微服务架构
- 具备前端+后端独立交付能力

### AI工程师
- 熟悉LLM API（OpenAI/Claude/Gemini）
- 有RAG系统构建经验
- Prompt工程实践
- 中文NLP经验优先

### 行业专家
- 5年以上行业经验（餐饮/美业/制造等）
- 熟悉行业痛点和工作流程
- 具备数字化转型推动经验
- 有产品思维优先
