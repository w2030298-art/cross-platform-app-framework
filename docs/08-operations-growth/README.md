## 8.3 冷启动策略（按阶段）

### Phase 1（0-3个月）— 种子期

**目标**：验证PMF（Product-Market Fit）

- 选1-2个最痛点场景，做MVP
- 定向找10-20个种子客户深度共创
- 通过种子客户反馈快速迭代
- 关键指标：NPS > 40，7日留存 > 30%

**种子客户获取路径**：
```
微信端 → 同城餐饮/美业商户（地推+社群）
小红书端 → 活跃品牌运营个人（社群邀请）
飞书端 → 飞书用户社区发帖（开发者论坛）
```

### Phase 2（3-6个月）— 验证期

**目标**：100个付费客户，验证商业模式

- 行业模板化，覆盖3-5个细分行业
- 启动内容营销（案例+教程）
- 申请各平台官方推荐位
- 关键指标：付费转化率 > 5%，月流失率 < 15%

**验证期增长路径**：
```
微信端 → 服务商渠道合作 + 行业社群口碑
小红书端 → 自运营账号种草 + KOL试用推荐
飞书端 → 飞书官方活动参与 + ISV生态合作
```

### Phase 3（6-12个月）— 增长期

**目标**：500个付费客户，MRR ¥50万

- 多平台联动（微信+小红书+飞书打通）
- 渠道合作体系建立
- AI能力全面接入
- 关键指标：NRR > 110%，CAC回收期 < 6个月

**增长期策略**：
```
微信端 → 行业服务商代理体系 + 老带新裂变
小红书端 → 品牌成功案例传播 + 行业白皮书
飞书端 → 行业解决方案标准化 + 区域代理商
```

### Phase 4（12-24个月）— 规模期

**目标**：2000个客户，MRR ¥200万

- 行业解决方案标准化
- 代理商体系建设
- 数据服务产品化
- 关键指标：毛利率 > 70%，净推荐值 > 50
